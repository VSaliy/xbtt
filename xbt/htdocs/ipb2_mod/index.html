XBT IPB2 Mod

Overview

XBT IPB2 Mod is a mod for Invision Power Board 2 and it's a frontend for XBT Tracker.
In forum views, it'll show statistics (seeders and leechers) for each topic that contains a .torrent attachment.
In topic views, it'll show statistics (seeders and leechers) for each .torrent attachment
The modified HTML templates are easily editable via the standard IPB2 template editor in the admin control panel, so you can modify the layout of the statistics.
